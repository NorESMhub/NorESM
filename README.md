# The Norwegian Earth System Model version 2.3 (NorESM2)
The **noresm2_3_develop** branch contains the NorESM2.3.X code and configurations, which is a<br />
technical development version of NorESM2. Bug fixes introduced in NorESM2.3 are answer changing,<br />
and this NorESM version does not have scientifically supported simulations.

The NorESM2.3 documentation: https://noresm-docs.readthedocs.io/en/noresm2.3/
